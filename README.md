# Domjur Public School Website

A modern, professional website for Domjur Public School featuring beautiful design, smooth transitions, and comprehensive information about the school.

## Features

- **Modern Design**: Professional color scheme with blue, gold, and green accents
- **Smooth Transitions**: Beautiful animations and transitions throughout
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hero slider, animated counters, gallery lightbox, and more
- **Comprehensive Pages**: All essential pages a professional school website needs

## Pages

1. **Home** (`index.html`) - Hero section, quick stats, programs overview, features, and news preview
2. **About Us** (`about.html`) - School history, mission, vision, and values
3. **Academics** (`academics.html`) - Academic programs, teaching methodology, and achievements
4. **Admissions** (`admissions.html`) - Admission process, requirements, fee structure, and application form
5. **Faculty** (`faculty.html`) - Meet the teaching team and their qualifications
6. **Facilities** (`facilities.html`) - School infrastructure, labs, sports facilities, and more
7. **News & Events** (`news.html`) - Latest news, upcoming events, and announcements
8. **Gallery** (`gallery.html`) - Photo gallery with filtering and lightbox
9. **Contact** (`contact.html`) - Contact information, contact form, and map

## Customization

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #f59e0b;
    --accent-color: #10b981;
    /* ... */
}
```

### Content
All content can be edited directly in the HTML files. Replace placeholder text and images with actual school information.

### Images
Currently using placeholder icons. Replace with actual images:
- Add images to a `images/` folder
- Update image paths in HTML
- For gallery, replace placeholder divs with actual `<img>` tags

## Usage
